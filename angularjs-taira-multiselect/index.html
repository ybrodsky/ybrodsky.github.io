Taira Multiselect Simple, Easy to use and Flexible Multiselect directive for AngularJS

Directive attributes

Model The scope array where you will be storing your selected items
Options The scope array that acts as a source for the select options
Settings An object containing the settings for the select

Basic setup

HTML 
  <taira-multiselect model="selected" options="dataSet" settings="settings"></taira-multiselect>
Controller 
  $scope.selected = [];
  $scope.dataSet = [
    {id: 1, name: 'Alejo', surname: 'Gerlingstein'},
    {id: 2, name: 'Topo', surname: 'Noesuficiente'},
    {id: 2, name: 'Vivian', surname: 'Redox'},
  ];
  $scope.settings = {
    displayField: ['name']        
  };
Rendered Html
$scope.selected 
  {{selected}}

Selected fields

By default Taira Multiselect pushes the whole object into the array of selected items. You can however specify which fields you want to push by changing the selectProperty setting.

Controller 
  $scope.settings = {
    displayField: ['name'],
    selectProperty: ['id', 'name']     
  };
Rendered Html
$scope.selected 
  {{selected2}}

Display options

The setting displayField is an Array of keys to be displayed in the options inside the select

Controller 
  $scope.settings = {
    displayField: ['name', 'surname']        
  };
Rendered Html

It is also posible to display a fixed string by changing the value of the settings displayField and displayFieldType

Controller 
  $scope.settings = {
    displayField: 'This is an option',
    displayFieldType: 'string'        
  };
Rendered Html

If you want to add some custom content before the displayed field -whether you are displaying a fixed string as the example above or an array of properties- you can by modifing the displayFieldPrependHTML setting.

Controller 
  $scope.settings = {
    displayField: ['name', 'surname']
    displayFieldPrependHTML: '<i class="fa fa-user"></i> ',
    showCheckbox: false   
  };
Rendered Html

In order to add a custom visual mark for the selected items, you can modify the selectedPrependHTML property from the settings object.
NOTE: In order for this to work you have to disable the checkboxes with the property showCheckbox: false

Controller 
  $scope.settings = {
    displayField: ['name', 'surname']
    showCheckbox: false,
    selectedPrependHTML: '<i class="fa fa-check"></i> '
  };
Rendered Html

Full list of settings

Attribute name Posible values Required What it does
displayField Array of values or a string Yes Sets the display property for the items in the select. An array of properties separated by a blank space or a fixed string.
displayFieldType 'string' | 'array' (default) No Sets the type of displayField. When 'string' is set, the items in the select will be showed with the fixed string defined in displayField
displayFieldPrependHTML Any piece of HTML (use common sense) No Sets a piece of HTML to be displayed in the items in the select before the displayField (whether displayFieldType is 'string' or 'array')
selectProperty Array of strings representing the options properties (default: []) No Sets the properties that will be pushed inside your selected items array. The default option '[]' pushes the whole object into the array.
btnClass Any class name (default: 'btn-primary') No Sets the class name to be applied to the button
btnText A string (default: 'Multiselect') No Sets the text the button will display
btnCountSelected true | false (default) No When set to true, the button text will be replaced by the count of selected items. 'None selected' when no item is selected and '{n} selected' otherwise.
menuClass Any class name No Sets the class name to be applied to the ul element of the list
open true | false (default) No When set to true, the list will be open on start.
selectAll true (default) | false No When set to true the 'Select all' option will be showed
unselectAll true (default) | false No When set to true the 'Unselect all' option will be showed
showCheckbox true (default) | false No When set to true a checkbox will be showed in every option
selectedPrependHTML Any piece of HTML (use common sense) No A piece of HTML to display before the displayField in every option when it is selected. In order for this to work, showCheckbox property should be false
